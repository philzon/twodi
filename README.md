# Twodi

![CI](https://github.com/philzon/twodi/workflows/CI/badge.svg?branch=master)

A 2D game engine implementation using SDL2.

# Build

## Prerequisites

* Bash
* Make
* G++
* SDL2

To build the project, execute the following command:

```txt
$ make clean all
```

Built binaries are placed in `bin/`.
